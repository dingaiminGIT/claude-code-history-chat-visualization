# 🚀 快速开始指南

## 一键启动（推荐）

```bash
# 1. 进入项目目录
cd claude-code-visualization

# 2. 运行安装脚本（只需运行一次）
./setup.sh

# 3. 启动工具
./run.sh
```

然后在浏览器访问: http://localhost:5000

## 手动启动

如果你更喜欢手动控制：

```bash
# 1. 创建虚拟环境
python3 -m venv venv

# 2. 激活虚拟环境
source venv/bin/activate

# 3. 安装依赖
pip install -r requirements.txt

# 4. 启动应用
python app.py
```

## 功能介绍

### 🏠 主页仪表板
- 查看对话统计（总数、项目数、会话数）
- 浏览最近的对话记录
- 快速了解项目分布

### 📚 对话历史
- 分页浏览所有对话记录
- 按时间倒序排列
- 一键复制对话内容
- 查看详细的会话信息

### 🔍 搜索功能
- 按关键词搜索对话内容
- 支持项目路径过滤
- 搜索结果高亮显示
- 实时搜索反馈

## 数据说明

工具会自动读取以下 Claude Code 数据：

- **历史记录**: `~/.claude/history.jsonl`
  - 包含对话的基本信息（时间、内容、项目）

- **调试日志**: `~/.claude/debug/`
  - 包含详细的会话调试信息（可选）

## 常见问题

### Q: 看不到任何对话记录？
A: 请确保：
1. 已经安装并使用过 Claude Code
2. `~/.claude/history.jsonl` 文件存在
3. 有过实际的对话记录

### Q: 工具启动失败？
A: 检查：
1. Python 版本是否 >= 3.7
2. 是否正确激活了虚拟环境
3. 依赖是否安装成功

### Q: 想要停止工具？
A: 在终端按 `Ctrl+C` 即可停止服务

## 技术支持

如果遇到问题，可以：
1. 检查终端输出的错误信息
2. 确认 Claude Code 配置目录存在
3. 重新运行安装脚本
