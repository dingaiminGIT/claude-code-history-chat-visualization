# Claude Code 对话可视化工具

一个用于可视化 Claude Code 对话历史的工具，让你能够方便地浏览、搜索和管理与 Claude 的对话记录。

## ✨ 功能特性

- 📊 **对话统计**: 展示对话总数、涉及项目、会话统计等
- 📝 **历史浏览**: 分页浏览所有对话记录，支持时间排序
- 🔍 **智能搜索**: 按关键词搜索对话内容，支持项目过滤
- 🗂️ **项目管理**: 按项目路径组织和筛选对话
- 📋 **快速操作**: 一键复制对话内容，查看详细信息
- 🎨 **美观界面**: 基于 Bootstrap 的现代化 Web 界面

## 🚀 快速开始

### 方法一：一键启动（推荐）

```bash
python3 start.py
```

### 方法二：手动启动

```bash
# 1. 安装依赖
pip3 install -r requirements.txt

# 2. 运行工具
python3 app.py
```

然后在浏览器中访问 `http://localhost:5000`

## 📁 数据源说明

工具会自动读取 Claude Code 的数据：
- **历史记录**: `~/.claude/history.jsonl` - 对话基本信息
- **调试日志**: `~/.claude/debug/` - 详细的会话日志
- **项目信息**: 自动提取项目路径信息

## 🛠️ 技术栈

- **后端**: Python 3.7+ + Flask
- **前端**: Bootstrap 5 + JavaScript
- **数据解析**: 自定义 JSON Lines 解析器
- **界面**: 响应式设计，支持移动端

## 🔧 开发扩展

### 添加新功能

2. 在 `app.py` 中添加路由和API
3. 在 `templates/` 中添加页面模板

### 自定义样式
- 修改 `templates/base.html` 中的 CSS
- 使用 Bootstrap 变量进行主题定制
